# YouTubeAnalyticsReport
Add your [YouTube API key](https://developers.google.com/youtube/registering_an_application) and your YouTube Channel ID into `.env` file.

```
vi .env

API_KEY="XXXXXXXX"
CHANNEL_ID="XXXXXXXX"
```
